My Projects

MyFitnessPro

MyFitnessPro is a dynamic fitness application built with React, designed to help users track and manage their workouts and nutrition. The app leverages React's component-based architecture to create an intuitive and responsive user interface. Additionally, I used Axios for seamless data fetching from external APIs, ensuring real-time updates and a smooth user experience. This project showcases my proficiency in modern JavaScript frameworks, API integration, and creating feature-rich, interactive web applications.

The Argentina National Football Club

This project is a tribute page dedicated to Lionel Messi, showcasing my skills in front-end development with a focus on design and styling. Built using HTML, CSS, and Sass, the project demonstrates the use of modern CSS techniques, including modular and maintainable styles with Sass. The design is clean and responsive, ensuring an optimal viewing experience on any device. This project highlights my ability to create visually appealing web pages while efficiently managing styles using Sass.

Porsche Website Clone

This project is a pixel-perfect clone of the official Porsche website, created to demonstrate my skills in front-end development. Using HTML, CSS, Bootstrap, and JavaScript, I replicated the design and functionality of the original site. The project focuses on responsive design, smooth animations, and intuitive navigation, all while leveraging Bootstrap's grid system and components to ensure a seamless user experience across devices. This clone highlights my ability to work with popular frameworks and deliver polished, professional web interfaces.
